# Doc Generator

## 用途

從 Python 檔案提取 module/class/function docstring，自動產生 Markdown 文件。

## Canonical 結構

- canonical script: `.agent/skills/doc-generator/scripts/doc_generator.py`

## 使用方式

```bash
python .agent/skills/doc-generator/scripts/doc_generator.py <file_path>
```
