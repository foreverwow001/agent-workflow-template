# Workflow References

這個目錄放的是 workflow 用的 reference 文件，不是新的 gate 契約來源。

## 目前內容

- [coordinator_research_skill_trigger_checklist.md](./coordinator_research_skill_trigger_checklist.md)
  - Coordinator 在 Research Gate 中判定 `deep-research` / `fact-checker` 的權威 checklist
- [engineer_skill_trigger_checklist.md](./engineer_skill_trigger_checklist.md)
  - Engineer 在實作前判定 `refactor` / `typescript-expert` / `python-expert` 的權威 checklist
- [workflow_skill_trigger_design_principles.md](./workflow_skill_trigger_design_principles.md)
  - 說明為什麼只有部分角色需要 checklist，其他角色暫不需要
- [workflow_skill_trigger_index.md](./workflow_skill_trigger_index.md)
  - workflow skill trigger 的導航索引，方便快速找到各角色的權威 checklist

## 使用原則

- 真正判定時，請回到各角色對應的 authoritative checklist
- `workflow_skill_trigger_index.md` 只做導航，不取代任何 checklist
- 若未來新增新的 role-specific trigger checklist，請同步更新本 README
